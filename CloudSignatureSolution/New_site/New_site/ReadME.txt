New_site
	This project contains a web application which allows
a new user to: register a new account, to log into the website
and complete personal details in order to request a digital 
certificate, to modify personal details, to generate a personal certificate. 
	Generate certificate method calls the Certificate
Authority and Installcertificate method calls a C programming
language dynamic library which has the role to modify some 
properties of the certificate, in order to set the developed key 
storage provider as the certificate key storage provider.
