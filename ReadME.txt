CloudDigitalSignature
	This solution contains all components needed for a digital signature cloud based architecture.
It contains a Certificate Authority, developed by OpenSSL library; a WebSite project which includes the signing
web service, a C# class library which consumes that web service and a C++ wrapper which makes available the
calling of the web service which is written in C#, by a C application; the KSP application and a simple application
which searches and enumerates the installed Key Storage Providers.
	Each project contains a ReadME file with a summar description.
